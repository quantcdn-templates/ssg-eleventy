# About

This is a sample Eleventy site deployed on QuantCDN.

## About Eleventy

Eleventy is a simpler static site generator. It transforms a directory of templates into HTML. It works with multiple template languages and requires minimal configuration.

## About QuantCDN

QuantCDN is a static web content delivery platform that provides fast, secure, and scalable hosting for static sites and JAMstack applications.
